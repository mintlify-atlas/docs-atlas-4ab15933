# Creating Abilities

Abilities are special traits that Pokémon have which provide effects during battle or in the overworld. They are defined in `PBS/abilities.txt`.

## Ability Entry Structure

Each ability is defined on a single line with comma-separated values:

```txt
ID,InternalName,Name,Description
```

## Real Examples

```txt
1,STENCH,Hedor,"Debido al mal olor que emana, al atacar al rival puede hacerlo retroceder."

5,STURDY,Robustez,"Evita que el rival pueda debilitarlo de un solo golpe cuando tiene los PS al máximo."

65,OVERGROW,Espesura,"Potencia los movimientos de tipo Planta del Pokémon cuando le quedan pocos PS."
```

## Field Definitions

<ParamField path="ID" type="number" required>
  Unique identifier for the ability. Must be sequential starting from 1.
</ParamField>

<ParamField path="InternalName" type="string" required>
  Internal code name in all uppercase. Used in scripts and PBS files.
  
  Example: `OVERGROW`, `BLAZE`, `INTIMIDATE`
</ParamField>

<ParamField path="Name" type="string" required>
  Display name shown to players in battle and menus.
  
  Example: `Espesura`, `Mar Llamas`, `Intimidación`
</ParamField>

<ParamField path="Description" type="string" required>
  In-game description explaining what the ability does. Enclosed in quotes.
</ParamField>

## Creating a New Ability

<Steps>

### Step 1: Find the next ID

Look at the last ability in `abilities.txt` and use the next sequential number.

### Step 2: Choose an internal name

Pick a descriptive, uppercase name:

```txt
320,FLAMESHIELD,Escudo Llama,"Reduce el daño de ataques de tipo Agua y aumenta el poder de movimientos de tipo Fuego."
```

### Step 3: Write a clear description

Explain what the ability does in simple terms:

```txt
321,TOXICTOUCH,Toque Tóxico,"Los movimientos de contacto tienen 30% de probabilidad de envenenar al objetivo."
```

### Step 4: Implement in scripts (optional)

For the ability to actually work, you'll need to add code in the scripts section. Basic abilities that modify stats or types may work without additional coding if they follow existing patterns.

</Steps>

<Warning>
  Adding an ability to `abilities.txt` only creates the database entry. To make it functional in battle, you must implement its effect in the game's scripts under `PokeBattle_Ability` or related files.
</Warning>

## Common Ability Patterns

### Type-Boosting Abilities

Abilities that increase damage of specific move types:

```txt
91,ADAPTABILITY,Adaptable,"Potencia aún más los movimientos cuyo tipo coincida con el del Pokémon."

89,IRONFIST,Puño Férreo,"Aumenta la potencia de los puñetazos."

185,STRONGJAW,Mandíbula Fuerte,"Su robusta mandíbula le confiere una mordedura potente."
```

### Speed Manipulation

```txt
33,SWIFTSWIM,Nado Rápido,"Sube la Velocidad cuando llueve."

34,CHLOROPHYLL,Clorofila,"Sube la Velocidad cuando hace sol."

95,QUICKFEET,Pies Rápidos,"Aumenta la Velocidad si sufre problemas de estado."
```

### Stat Prevention

```txt
29,CLEARBODY,Cuerpo Puro,"Evita que bajen sus características a causa de movimientos o habilidades de otros Pokémon."

73,WHITESMOKE,Humo Blanco,"El humo blanco que lo protege evita que otro Pokémon le baje las características."

52,HYPERCUTTER,Corte Fuerte,"Evita que el rival le baje el Ataque."
```

## Hidden Abilities

Hidden abilities are defined the same way but are designated in `pokemon.txt`:

```txt
# In pokemon.txt
[1]
Name=Bulbasaur
Abilities=OVERGROW
HiddenAbility=CHLOROPHYLL
```

<Note>
  Hidden abilities are typically rarer and obtained through special means like Max Raid Battles, special distributions, or breeding.
</Note>

## Multi-Effect Abilities

Some abilities have multiple effects:

```txt
47,THICKFAT,Sebo,"Gracias a la gruesa capa de grasa que lo protege, reduce a la mitad el daño recibido por ataques de tipo Fuego y Hielo."

87,DRYSKIN,Piel Seca,"Pierde PS si hace sol y los recupera si llueve o recibe un movimiento de tipo Agua. Los movimientos de tipo Fuego, le hacen más daño de lo normal."
```

### Ability Copy/Change

```txt
150,IMPOSTER,Impostor,"El Pokémon se transforma en el que tiene enfrente."

152,MUMMY,Momia,"Contagia la habilidad Momia al rival que entre en contacto con él."
```

## Ability Naming Conventions

<Tip>
  When creating custom abilities:
  - Use descriptive internal names (e.g., `IRONBARBS`, not `ABILITY1`)
  - Keep display names concise (2-3 words max)
  - Write clear descriptions that explain the effect without being too wordy
  - Test thoroughly with different Pokémon and battle scenarios
</Tip>

## Testing Abilities

<Steps>

### Use Debug mode

Press F8 in-game to open debug menu and test abilities.

### Create a test Pokémon

Assign your new ability to a test Pokémon in `pokemon.txt`:

```txt
[999]
Name=TestMon
InternalName=TESTMON
Abilities=YOURNEWABILITY
```

### Battle test

Use the debug battle option to verify the ability works as intended in various scenarios.

</Steps>
