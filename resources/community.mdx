## Overview

The Pokémon Essentials community is vibrant, helpful, and full of talented developers. Whether you're just starting out or you're an experienced maker, there are plenty of resources and communities to help you on your journey.

<Note>
Pokémon Essentials BES is particularly popular in the Spanish-speaking community, so you'll find excellent resources in both English and Spanish.
</Note>

## Getting Help

### Before Asking for Help

Before posting questions, try these steps:

1. **Read the documentation** - This documentation covers most common scenarios
2. **Search existing threads** - Your question might already be answered
3. **Check error logs** - Look in your game folder for error files
4. **Try the FAQ** - See our [FAQ page](/resources/faq) for common questions
5. **Review troubleshooting** - Check [Troubleshooting](/resources/troubleshooting) for solutions

### When Asking for Help

<Info>
Provide these details when asking for help:
</Info>

- **What you're trying to do** - Describe your goal
- **What's actually happening** - Describe the problem
- **Error messages** - Copy the full error text
- **What you've tried** - List troubleshooting steps taken
- **Relevant code** - Share the specific code causing issues
- **Essentials version** - Specify you're using Essentials BES

### Contributing to BES

If you want to contribute to Pokémon Essentials BES:

1. **Report Bugs** - Submit detailed bug reports
2. **Suggest Features** - Propose improvements
3. **Fix Issues** - Submit pull requests for bugs
4. **Update Documentation** - Help improve this documentation
5. **Share Gen 9 Fixes** - Contribute completed implementations

<Note>
When contributing, maintain the BES philosophy: add only what's strictly necessary and integrate with original code for compatibility.
</Note>

## Code of Conduct

When participating in the community:

✅ **Do:**
- Be respectful and patient
- Give credit for resources used
- Search before asking duplicate questions
- Share knowledge when you can
- Follow community guidelines

❌ **Don't:**
- Share copyrighted materials without permission
- Ask for direct game downloads
- Spam or advertise excessively
- Demand help or become aggressive
- Share ROM hacks or pirated content
