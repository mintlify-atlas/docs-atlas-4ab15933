## About This Project

Pokémon Essentials BES is an unofficial fork of Pokémon Essentials v16.2 and the successor to the "Base de Pira". This project brings together the work of countless talented developers, artists, and community members.

<Note>
If any credits are missing, please report them so they can be added. This is a community effort and everyone deserves recognition.
</Note>

## Using This Project

When using Pokémon Essentials BES in your project, please include appropriate credits:

### Minimum Required Credits

```
Pokémon Essentials BES
- Pira (Base compilation)
- Clara (Co-compiler)
- Pokémon Essentials team (Flameguru, Poccil, Maruno)

Sprites: Smogon Sprite Project and contributors
Scripts: [See full credits in Créditos.txt]

Pokémon © Nintendo, Game Freak, The Pokémon Company
```

### Full Credits

For a complete credits implementation, we recommend:

1. Including the full `Créditos.txt` file in your project
2. Creating an in-game credits sequence
3. Adding credits to your game's README or documentation
4. Crediting any additional resources you add

<Note>
Always credit scripters and sprite artists for the resources you obtain from them. The community thrives on proper attribution.
</Note>

## Contributing to Pokémon Essentials BES

If you've contributed to this project and your name is missing, or if you notice missing credits, please:

1. Report the issue to the project maintainers
2. Join the [Community](/resources/community) discussions
3. Submit corrections or additions

This project is a collaborative effort, and everyone's contributions matter.
